# MovieHub

## Документация для Telegram-бота

### Описание проекта
Это Telegram-бот, предоставляющий пользователям персонализированные рекомендации фильмов на основе их предпочтений по жанрам. Пользователь взаимодействует с ботом через кнопки и команды, выбирает жанры и получает список фильмов с высокими рейтингами. Проект сочетает интерактивность и удобство, предоставляя рекомендации в удобном формате.
